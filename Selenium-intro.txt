Selenium Overview

Selenium is a powerful browser automation tool. It supports various browsers like Firefox, Chrome, 
Internet Explorer, Edge, Safari. Webdriver is the heart of Selenium Python. 
It can be used to perform various operations like automating testing, 
perform operations on-webpage elements like close, back, get_cookie, get_screenshot_as_png, get_window_size to name a few.
Some common use-cases of using selenium for web scraping are automating a login, submitting form elements,
handling alert prompts, adding/deleting cookies, and much more. It can handle exceptions as well.

Installation

pip install selenium
